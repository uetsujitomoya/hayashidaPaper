\section{結論}

本研究では,word2vecによって得られた単語のベクトルから1文毎の文ベクトルを算出し,SVMを用いて機械学習を行い,クライエントの発言1文毎の自動カテゴリ分類を行い,会話の流れの可視化システムにおける辞書に基づいたカテゴリ分類と比較して,高い分類精度を実現した.

しかし、前章で述べたとおり,ユーザーの手動修正の負担がかからない分類精度には及んでおらず,分類精度をさらに高める必要がある.さらにもっと多くの未知のクライエントの発言文に対して分類精度を検証する必要がある.次節に今後の課題について述べる.

\section{今後の課題}

本研究で得られたことを踏まえて,今後検討するべき課題について述べる.
\begin{itemize}
\item 「あの人の仕事を否定したくありません.」というような文を正しく分類するために,文の係り受け解析を行い,「あの人」の「仕事」であることから「愛」のカテゴリであると分類できるようにすること
\item クライエントの前の発言に遡らないとカテゴリが把握できない文や,カウンセラーからの質問に遡らないとカテゴリが把握できない返事などの短い文を正しく分類するために,クライエントの発言1文毎の分類ではなく,カウンセラーの質問やクライエントの前の発言を含めた,会話のブロック毎の分類
\item 主語が抜けている文や,「あの人」のように主語の特定が困難である文を正しく分類するために,その前の会話に遡り主語を補うことができるかの検討
\item さらなる未知のクライエントの発言文に対する分類と,「交友」カテゴリに属するクライエントの発言文に対する分類の検討
\item 教師付きデータとして,Yahoo!知恵袋の質問文以外のテキストデータの検討
\item 学習データ数による分類の正答率の推移の検討
\item 分類カテゴリに「自己」,「スピリチュアル」の追加
\end{itemize}
